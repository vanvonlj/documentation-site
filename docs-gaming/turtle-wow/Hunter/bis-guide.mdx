# Hunter Best in Slot Guide

Comprehensive Best in Slot (BIS) gear list for Hunters in Turtle WoW. Use the filters and sorting on each column to find the best gear for your slot, source, or stat priority.

:::tip How to use this table
- **Filter by slot**: Type a slot name (e.g. "Head", "Ranged") in the Slot filter to see all items for that slot
- **Filter by source**: Type a location (e.g. "BWL", "AQ40", "MC", "Naxx", "PvP") to see drops from that source
- **Sort by stat**: Click any stat column header to sort ascending/descending
- **Sort by Score**: The AP Equiv column ranks items by overall DPS value
:::

### Stat Weights

The **AP Equiv** score uses these approximate stat weights to rank items:

| Stat | Weight | Reasoning |
|------|--------|-----------|
| **AGI** | 2.0 per point | 1 RAP + crit contribution |
| **STR** | 1.0 per point | 1 melee AP |
| **Crit %** | 25 per 1% | High value for burst |
| **Hit %** | 15 per 1% | Critical until 9% cap |
| **AP** | 1.0 per point | Direct damage increase |

:::info
These are approximate weights. Your actual stat values may vary based on current gear, buffs, and encounter type. Ranged weapon scores include weapon DPS contribution.
:::
